# SSRF Demo

## Instructions to run the project
1. Install docker desktop.
2. Clone this project/download zip file of this project.
3. Open the docker (keep it open).
4. Run `docker-compose up --build -d` command in the project root directory.
5. You can open `http://localhost` in your browzer to see the website.
6. You can use your browser/burp suite to get sensitive info from the website
